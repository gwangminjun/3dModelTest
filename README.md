# 3dModelTest
GLB/GLTF 형식의 3D 건물 모델을 불러와 균열·붕괴·침하·노화 변형 효과와 날씨 시뮬레이션을 적용하는 웹 애플리케이션.

---

## 개선 가능 사항

### 즉시 구현 가능 (난이도 낮음)

| # | 기능 | 설명 | 상태 |
|---|------|------|------|
| 1 | **스크린샷 저장** | `renderer.domElement.toDataURL()`로 현재 화면을 PNG로 다운로드 | ✅ 완료 |
| 2 | **카메라 시점 프리셋** | 정면/측면/상단 버튼으로 고정 시점으로 즉시 이동 | ✅ 완료 |
| 3 | **시뮬레이션 재생 배속** | 0.5x / 1x / 2x / 4x 배속 선택 옵션 추가 | ✅ 완료 |
| 4 | **수동값 → 시뮬레이션 목표 복사** | 현재 수동 변형값을 시뮬레이션 목표값에 한 번에 복사하는 버튼 | ✅ 완료 |

### 중간 난이도

| # | 기능 | 설명 | 상태 |
|---|------|------|------|
| 5 | **상태 저장/불러오기** | `localStorage`에 날씨·강도·변형값 저장, 다음 방문 시 복원 | ✅ 완료 |
| 6 | **미니맵 뷰 전환** | 레이블 클릭으로 Side / Top / Front View 순환 | ✅ 완료 |
| 7 | **날씨 자동 변형 범위 확장** | `calcAutoTargets()` 최대값을 100 → 500으로 확장, 날씨 효과 더 극적으로 표현 | ✅ 완료 |
| 8 | **모델 정보 표시** | 로드된 GLB의 메시 수, 버텍스 수 패널에 표시 | ✅ 완료 |

### 큰 변경

| # | 기능 | 설명 |
|---|------|------|
| 9 | **다중 날씨 조합** | 비 + 강풍처럼 두 날씨를 동시에 선택 (라디오 → 체크박스 구조 변경 필요) | ✅ 완료 |
| 10 | **키프레임 시뮬레이션** | 여러 단계(시간·목표값 쌍)를 정의하는 다단계 시나리오 기능 | ✅ 완료 |
